# Research References

## Generative UI

- **A2UI v0.9** (Google, April 2026) — Open declarative protocol for agent-driven interfaces. https://a2ui.org
- **AG-UI Protocol** (CopilotKit, 2025) — Event-based agent-to-frontend communication standard. https://docs.copilotkit.ai/ag-ui-protocol
- **Thesys C1** (April 2025) — World's first Generative UI API. https://thesys.dev
- **Vercel AI SDK 3.0** (2024) — Generative UI with React Server Components. https://vercel.com/blog/ai-sdk-3-generative-ui
- **Renderify** (Feb 2026) — Runtime engine for rendering LLM-generated UI instantly in the browser.
- **OpenGenerativeUI** (CopilotKit) — Open-source reference implementation. https://github.com/CopilotKit/generative-ui

## Educational Animations

- **LAVES** (Baidu, Feb 2026) — LLM-based multi-agent system for educational video generation. 1M+ videos/day, >95% cost reduction. arXiv:2602.11790
- **LLM2Manim** (April 2026) — Pedagogy-aware AI generation of STEM animations. d=1.64 satisfaction effect size vs. slides. arXiv:2604.05266
- **Training and Agentic Inference Strategies for LLM-based Manim** (April 2026). arXiv:2604.18364
- **Manim for STEM Education** (2025). arXiv:2510.01187
